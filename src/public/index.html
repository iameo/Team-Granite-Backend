Team Granite Dockarized User Management App

Testing the App using Postman

Get All Users: In Postman open your workspace,

  • Set your HTTP request to GET.
  • In the request URL field, input link with /users at the end e.g http://localhost:5000/users
  • Click Send
  • You will see ‘200 OK’ Message if it runs successfully
  • There should be a json file result of the users in the body which indicates that your test has run successfully.

Testing the POST request to add a user

Get A Single User: In the workspace,

  • Set your HTTP request to POST.
  • In the request URL field, input link with /user at the end e.g http://localhost:5000/user
  • In the body you will have to fill the required information
  • Click Send
  • You will see ‘200 OK’ Message if it runs successfully
  • There should be a json file result of the users in the body which indicates that you have added your user successfully.
